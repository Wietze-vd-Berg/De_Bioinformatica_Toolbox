RNA-analyse met Salmon

RNA-sequencing is een techniek waarmee de hoeveelheid RNA in een monster wordt gemeten.
Deze website dient als een efficiënte manier om RNA-seq data analyseren met behulp van Salmon,
een krachtige tool voor het kwantificeren van de genexpressie, het tot uitdrukking brengen van bepaalde genen. [RNA-Seq: a revolutionary tool for transcriptomics]

Salmon is een transcriptkwantificatie-tool, wat betekent dat het berekent hoeveel van elk RNA-transcript aanwezig is. [Salmon website]
Een transcript is een afgeschreven kopie van een gen, die later kan worden vertaald naar een eiwit.
Salmon gebruikt een probabilistisch model, wat inhoudt dat het wiskundige waarschijnlijkheden gebruikt om meetfouten te corrigeren.

Een groot voordeel van Salmon is pseudo-uitlijning.
Bij traditionele methoden worden RNA-sequenties volledig uitgelijnd tegen een referentiegenoom, wat veel rekenkracht kost. [A survey of best practices for RNA-seq data analysis]
Pseudo-uitlijning daarentegen bepaalt snel of een RNA-fragment overeenkomt met bekende genen zonder de exacte positie te bepalen. [Near-optimal probabilistic RNA-seq quantification]
Dit maakt de analyse vele malen sneller en efficiënter.
